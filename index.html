Einfacher PyScript Rechner

Gib zwei Zahlen ein und klicke auf "Berechnen" um das Ergebnis der Addition zu sehen:

from js import document def calculate(): num1 = float(document.getElementById("num1").value) num2 = float(document.getElementById("num2").value) result = num1 + num2 document.getElementById("result").innerText = f"Das Ergebnis ist: {result}"
